## 一、指向指针的指针
### 1、指向指针的指针：
* 指针变量也是变量，占据一定的内存空间，有地址所以可以使用一个指针指向它，有称为二级指针。
* 可以通过“**”声明一个二级指针
* 理解：当需要知道指针p1的地址但是不知道，指针p2知道，可以先通过指针p2去拿指针p1，p2称为指向指针的指针

```
double num =11;
double *p = &num;
double *pp = &p;
```
##### 说明
* 运行效果：编译失败
* 失败原因：double的长度为8个字节，指针的地址为4个字节，将指针p的地址作为指针pp值长度不符合


```
double num =10;
double *pN =&num;
double **ppN =&pN;
```
