#### 1、理解size_t
* size_t表示C中任何对象所能达到的最大长度，是无符号整数。
* size_t用作sizeof操作符的返回值类型，也是很多函数的参数类型，包括malloc和strlen。
* 在声明像字符串或数字索引这样的长度变量时用size_t是好的做法，还经常用于循环计算器、数组索引，有时候还用在指针算术运算上。

```
#ifndef _SIZE_T
#define _SIZE_T

typedef unsigned int size_t;
#endif
```
